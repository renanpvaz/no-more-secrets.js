no-more-secrets.js

An attempt at creating the decryption effect seen in the 1992 movie Sneakers, but with JavaScript. Inpired by this.

How to start using it: download the latest version, add it to your project and call the exposed function nms(element = document.body).

Optionally, you can pass as second argument a configuration object that is merged with the default values: 


  {
    maxUpdateInterval: 200, // maximum update interval time (in milliseconds)
    minUpdateInterval: 50, // minimum update interval time (in milliseconds)
    maxUpdates: 20, // number of updates until decryption ends
    extraDelay: 100 // small addition with .1 chance of happening to some characters
  }
