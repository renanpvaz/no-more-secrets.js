no-more-secrets.js

An attempt at recreating the decryption effect shown in the 1992 movie Sneakers, but with JavaScript. Inpired by this.

How to start: download the latest version, add it to your project and call the exposed function nms(element, config). The target element defaults to body.

Optionally, you can pass as second argument a configuration object which will be merged with the default values: 


  {
    maxUpdateInterval: 100, // maximum update interval time
    minUpdateInterval: 25,  // minimum update interval time
    maxUpdates: 50,         // number of updates until character decryption ends
    extraDelay: 50,         // small interval addition with .1 chance of happening
    initialDelay: 100,      // delay before the decryption effect start
  }

Notes: the effect will only work correctly with monospaced fonts; the more text nodes you have the slower it becomes.
